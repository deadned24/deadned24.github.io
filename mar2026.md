# You can customize Mafia's login logo, and it's easier than you might think.
*article by **DeadNed** (#1909053)*

Did you know that you can change the login logo? "Sure," you say, "Just fire up the ol' Eclipse IDE, track down the images in the startup frame, do the changes and compile, mafia is open source after all...". But it's even easier than that! Mafia has a preference, `loginWindowLogo`, to store the login window logo. It ships with lantern.gif, as the default and also limeglass.gif (the previous default), as a fun alternate. You could type `set loginWindowLogo=limeglass.gif` in the gCLI to change it for next login, and to change back, `set loginWindowLogo=lantern.gif`. 

But wait, there's more! You can set the login logo to *any* image (png/jpg/gif), not just limeglass and lantern. The image needs to be in your images folder. It should be small and roughly the size you want for the login window, since the window will stretch around it and won't scale or scroll.  The syntax is the same, `set loginWindowLogo=FILE`, where FILE is the file with extension For example: `set loginWindowLogo=9xcCy82.png`.

Changing the logo is a great way to personalize mafia, even if you're not graphically inclined. KOL offers several tools for creating unique images such as the Meme generator (via the Intergnat's Internet Meme Shop), bazooka fish bubble gum, and VIP Clan Photo Booth's "take a group photo" to produce a login logo that fits game's style, but a picture of your cat or favorite bird is an option too!
